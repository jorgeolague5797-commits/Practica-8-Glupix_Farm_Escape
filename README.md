# GLUPIX FARM ESCAPE

## Descripción general

**GLUPIX FARM ESCAPE** es un juego estilo voxel inspirado en Crossy Road. El jugador controla a **Glupix**, un alien que se estrelló en una granja mientras intentaba llevarse unas vacas. Ahora debe escapar avanzando por un camino infinito lleno de vacas, manadas de pollos, obstáculos de granja y eventos especiales.

El objetivo es avanzar lo más lejos posible, aumentar el score, recoger artefactos espaciales y superar la mejor puntuación histórica.

## Controles de movimiento

- **W / Flecha arriba:** mirar hacia adelante.
- **A / Flecha izquierda:** mirar a la izquierda.
- **S / Flecha abajo:** mirar hacia atrás.
- **D / Flecha derecha:** mirar a la derecha.
- **Espacio:** avanzar o saltar en la dirección en la que Glupix está mirando.
- **ESC / P:** abrir o cerrar el menú de pausa.

## Coleccionables

Durante la partida aparecen artefactos alienígenas raros cada cierta cantidad de puntos. Estos objetos representan piezas y recuerdos espaciales que Glupix perdió al estrellarse.

Coleccionables incluidos:

- **Mini OVNI**
- **Planeta Tierra**
- **Mini alien**
- **Cristal espacial**
- **Núcleo de energía**

Cada coleccionable aumenta el contador de **Artefactos** en el HUD y suma puntos extra al score. Para que se sientan únicos, aparecen con menor frecuencia y solo puede haber un artefacto activo al mismo tiempo.

## Base de datos local

El juego utiliza una base de datos local **SQLite** para guardar las puntuaciones.
