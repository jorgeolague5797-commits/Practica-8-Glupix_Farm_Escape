# Pr-ctica-8-Glupix_Farm_Escape
GLUPIX FARM ESCAPE es un juego voxel estilo Crossy Road donde controlas a Glupix, un alien que debe escapar de una granja infinita. Avanza, esquiva vacas y manadas de pollos, recoge artefactos espaciales raros, activa eventos especiales y busca superar tu mejor puntuación histórica.
